İSTERLER:

- Kullanıcı giriş ve kayıt modülü 	: Giriş başlıklı sayfada birlikte düzenlendi
- Saat ve tarih gösterimi		: Footer'a eklendi
- Navigation bar			: Sticky olarak düzenlendi
- Menüler burger menü			: Nav bar responsive yapıda dikey açılan menü olarak düzenlendi
- Eğitimler kısmına link		: Nav bar "Eğitimler" kısmına link eklendi
- İletişim kısmı			: İletişim bilgileri ve mesaj gönderme formu düzenlendi
- Standart kare tek yapılar olmasın	: Tasarımda genel olarak oval / b- radius : 5px öğeler kullanıldı
- İşle ilgili renk seçilecek		: İTO kurumsal renklerine uygun yapıda tasarım uygulandı
- Yazılar tek bir font olacak		: Tüm yazılar 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif yapıldı
